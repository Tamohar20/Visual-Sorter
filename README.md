# 🚀 Visual Sorter

An interactive **Sorting Algorithm Visualizer** built using HTML, CSS, and JavaScript.

This project helps in understanding how different sorting algorithms work through **real-time animations and visual representation**.

---

## 🌟 Features

- 🎯 Visualize multiple sorting algorithms
- 📊 Real-time comparisons & swaps counter
- ⚡ Speed control (Slow → Turbo)
- 🎲 Random array generation
- ✍️ Custom array input
- 🌙 Light/Dark mode toggle
- 📈 Algorithm performance comparison
- 💾 Save & Load array feature
- 🧠 Step-by-step execution mode

---

## 🧮 Algorithms Implemented

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Counting Sort

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Advanced UI + Themes)
- JavaScript (Core Logic + Animations)
